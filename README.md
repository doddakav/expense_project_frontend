Expense Tracker Frontend
A Streamlit frontend application for tracking and managing expenses with FastAPI backend integration.
________________________________________
Features
•	Add Expenses
•	View Expenses
•	Update Expenses
•	Delete Expenses
•	Search Expenses
•	Sort Expenses
•	Interactive Expense Tables
•	Backend API Integration
________________________________________
Tech Stack
•	Streamlit
•	Pandas
•	Requests
•	Streamlit Community Cloud
•	FastAPI Backend

Installation
Clone Repository
git clone https://github.com/doddakav/expense_project_frontend.git
Install Dependencies
pip install -r requirements.txt
________________________________________
Streamlit Secrets
Add Streamlit secrets:
server_url="https://your-backend-url.onrender.com"
________________________________________
Run Frontend Locally
streamlit run app.py
